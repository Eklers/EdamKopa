This is CSS training part

Main idea:
  • create 3 div elements (for comfort put them in one div);
  • use css to transform (use -webkit and -moz for different browsers);
  • do 'little' math;
  • add shadings for 3D view.

It would be complicated to place in correct place (pixel by pixel), because of div locations and transformations. Here solved with margin.
P.S. - code is a bit different from real picture.

.cube {
position: relative;
top: 200px;
}
.rightFace, .leftFace, .topFace div {
padding: 10px;
width: 180px;
height: 180px;
}
.rightFace, .leftFace, .topFace {
position: absolute;
}
.leftFace {
-webkit-transform: skewY(30deg);
background-color: #ccc;
}
.rightFace {
-webkit-transform: skewY(-30deg);
background-color: #ddd;
left: 200px;
}
.topFace div {
-webkit-transform: skewY(-30deg) scaleY(1.16);
background-color: #eee;
font-size: 0.862em;
}
.topFace {
-webkit-transform: rotate(60deg);
top: -158px;
left: 100px;
}
